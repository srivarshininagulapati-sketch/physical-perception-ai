# Physical Perception AI – Afferens Vision System

This project is an AI-based Physical Perception system that uses live camera input and Afferens API to detect objects in the real world and make intelligent decisions.

## 🚀 Features
- Real-time object detection using Afferens API
- Live camera-based perception
- Detects humans and objects using COCO-SSD model
- Simple Python-based implementation

## 🧠 Concept
AI agents are powerful in reasoning but cannot directly see the real world.  
This project bridges that gap using live visual perception.

## ⚙️ Tech Stack
- Python
- Requests library
- Afferens API
- COCO-SSD model

## 🔄 Flow
Camera → Afferens API → Object Detection → Decision Engine → Action
